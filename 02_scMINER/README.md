Three steps are included in the scMINER analysis:

1) Prepare scMINER Input (./01_scMINER_input)
    A. 01_sample_list.txt: four columns--sample ID, species, Input Directory (output files of quality control) and Output Directory.
    B. 02_get_Rmd.pl: generate the .Rmd file from the template file '02_scRNASeq_MICA_Input.Rmd'.
    C. 02_scRNASeq_MICA_Input.Rmd: the template file for scMINER input file.
    D. 03_run_scMINER_Input.sh: generated by '02_get_Rmd.pl', including the .Rmd files of all samples.

2) Run MIE (Mutual Information Estimation, ./02_run_MIE)
    A. 00_sample_list.txt: Input sample list for scMINER, both MIE and MICA. Four columns were included: Sample ID, Species, Output Directory and Qualified Cell Numbers of the sample.
    B. 01_prepare_MIE.pl: generate the scripts of MIE. All scripts will be in current folder.
    C. 02_run_MIE.sh: run all MIE scripts.

3) Run MICA (Mutual Information Clustering Analysis, ./03_run_MICA)
    A. 01_prepare_MICA.pl: generate the scripts of MICA. All scripts will be in current folder.
    B. 02_run_MICA.sh: generate the scripts of MICA. All scripts will be in current folder.
    C. 03_pool_figures.sh: copy all scMINER Clustering figures into one folder.
